Web-based minigame to practice basic Vim commands.

Try out the minigame here: https://vimulator.netlify.app/

## Tech Stack
Frontend: React

Backend: Express

Database: Supabase


## How to Run
1. Clone the repo.
2. Inside the frontend folder, run:
   
   `npm install`
   
   `npm run dev`
3. Inside the backend folder, run:

   `npm install`

   `npm install --save-dev nodemon`
   
   `npm run dev`
